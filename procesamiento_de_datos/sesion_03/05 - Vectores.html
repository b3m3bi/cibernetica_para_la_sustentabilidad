05 - Vectores

Introducción

Ya vimos que en R podemos trabajar con números o cadenas de caracteres individuales, sin embargo, generalmente vamos a trabajar con conjuntos de datos y no con valores individuales. En R la forma más básica de agrupar varios valores es usando vectores en R. En este notebook exploramos qué son y cómo se crean y usan los vectores.

Objetivos

  • Crear y utilizar vectores para almacenar conjuntos de datos

¿Qué es un vector?

Un vector es una colección ordenada de valores del mismo tipo. Por ejemplo podríamos tener un vector para guardar números o un vector para guardar cadenas de texto.

Para crear un vector usamos la función c() (la «c» significa “combinar”). Dentro de la función c() colocamos los elementos que queremos incluir en nuestro vector cada uno separado por una coma. Por ejemplo:

temperaturas <- c(23.5, 19.2, 21.8, 20.2)

alumnos <- c("Jimena", "Gabriela", "Paulina", "Pamela", "Ana", "Alan", "Regina", "Linda")

También podemos hacer vectores usando los valores que tienen otras variables:

especie01 <- "Homo neanderthalensis"
especie02 <- "Homo habilis"
especie03 <- "Homo sapiens"

especiesHomo <- c(especie01, especie02, especie03)

print(especiesHomo)
[1] "Homo neanderthalensis" "Homo habilis"          "Homo sapiens"

Acceder a los elementos de un vector

Los vectores son como un casillero, el cual tiene varias cajas en las cuales podemos guardar nuestros objetos. Así como los casilleros están numerados también los elementos de un vector están numerados, a este número se le conoce como índice. El índice me permite acceder a los objetos de acuerdo a su posición en el vector, para ello puedo poner el nombre el vector y luego entre corchetes [] colocar el número del índice del elemento al cual quiero acceder:

colores <- c("azul", "rojo", "verde", "amarillo")

colores[1]
[1] "azul"
colores[3]
[1] "verde"

Nota la primera posición tiene el índice 1. Esto contrasta con la mayoría de los lenguajes de programación donde generalmente los índices comienzan con el 0.

Operaciones aritméticas con vectores

Podemos hacer operaciones aritméticas con vectores numéricos. Por ejemplo, si queremos aumentar en 10 a cada valor de un vector podemos hacer lo siguiente:

numeros <- c(2, 4, 8, 10)

numeros + 10
[1] 12 14 18 20

Nota como R aplica la operación a cada elemento individual. Podemos aplicar cualquier otra operación que queramos:

numeros * 2
[1]  4  8 16 20
numeros / 4
[1] 0.5 1.0 2.0 2.5
numeros ^ 2
[1]   4  16  64 100

También podemos hacer operaciones que utilicen dos vectores:

primeros <- c(5, 10, 15)
segundos <- c(1, 2, 3)

primeros - segundos
[1]  4  8 12

Nota como estamos haciendo operaciones entre elementos que tienen la misma posición. Es decir R alinea los elementos del primer vector con los elementos del segundo vector y realiza las operaciones entre pares. En el ejemplo de arriba realiza las restas 5 - 1, 10 - 2 y 15 - 3.

Algunas funciones úitles con vectores numéricos

Existen muchas funciones útiles que podemos usar con vectores, más adelante veremos varias de ellas. Aquí vemos solo 4 de ellas.

La función mean() nos da el promedio de un vector:

temperaturas <- c(23.5, 19.2, 21.8, 20.2)

mean(temperaturas)
[1] 21.175

La función min() nos da el valor mínimo de un vector:

temperaturas <- c(23.5, 19.2, 21.8, 20.2)

min(temperaturas)
[1] 19.2

La función max() nos da el valor máximo de un vector:

temperaturas <- c(23.5, 19.2, 21.8, 20.2)

max(temperaturas)
[1] 23.5

La función length() nos da la longitud de un vector (el número de elementos que tiene):

temperaturas <- c(23.5, 19.2, 21.8, 20.2)

length(temperaturas)
[1] 4

Concatenar vectores de cadenas de texto

También hacer algunas cosas interesantes con vectores de texto, por ejemplo la función paste() puede concatenar los elementos en la misma posición de dos vectores:

primerNombre <- c("Pepito", "Luis", "Juan")
segundoNombre <- c("Grillo", "Miguel", "Gabriel")


nombresCompletos <- paste(primerNombre, segundoNombre)

print(nombresCompletos)
[1] "Pepito Grillo" "Luis Miguel"   "Juan Gabriel"

Pero también con paste() podemos agregarle una misma cadena a todos los elementos de un vector:

saludos <- paste("Hola", nombresCompletos)
print(saludos)
[1] "Hola Pepito Grillo" "Hola Luis Miguel"   "Hola Juan Gabriel"
